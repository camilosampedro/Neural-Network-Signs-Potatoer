# Neural-Network-Signs-Potatoer
OpenCV neural network that recognize transit signals

## Requirements for running
 - `Python 3.5.x`
 - `OpenCV 3.x`
### Available with pip
 - `scikit-learn 0.18.x`
 - `scikit-image 0.12.x`
 - `numpy 1.11.x`
 - `matplotlib 1.5.x`

## Execution
Run `main.py` file with `train` argument:

```bash
python main.py
```
